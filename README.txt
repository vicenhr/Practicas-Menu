Este proyecto presenta un menú en HTML que enlaza a cuatro prácticas desarrolladas durante el curso de Programación Web en la Unidad 2.

La primera práctica consiste en la creación de un currículum vitae en formato web, permitiendo plasmar información personal y profesional de manera organizada. La segunda práctica expande este concepto con una estructura más compleja que incluye un índice principal, una sección sobre la historia de la web y otra con el CV.

En la tercera práctica, el proyecto se diversifica con un conjunto de páginas interconectadas, organizadas en carpetas de css, html y multimedia, donde se presentan aspectos personales como intereses, hobbies y un video de presentación. Finalmente, la cuarta práctica mantiene esta organización pero cambia el enfoque hacia una guía turística del estado de Puebla, integrando elementos visuales y textuales para brindar información relevante sobre la región.
